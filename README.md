# Charity Donation Platform

## Project Description
The Charity Donation Platform is a web application that allows users to support charitable causes through online donations. The platform connects donors with verified organizations, making the donation process simple, transparent, and accessible.

Users can:
- Browse available charity campaigns
- View detailed information about each cause
- Make secure donations
- Track donation history
